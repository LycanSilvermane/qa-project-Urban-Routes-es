Proyecto de automatización de pruebas utilizando selenium

Descripción: Este es un proyecto en el cual se automatizan pruebas para una aplicación de alquiler de transporte llamada Urban Routes

Tecnologías: Se utilizaron las tecnologias Pytest, Pycharm, Google Chrome, Selenium y el lenguaje de programacion Python

Instrucciones: Para realizar estas pruebas es necesario instalar los paquetes Pytest y Selenium, la información utilizada para completar los formularios se encuentra en el documento data.py y las mismas se pueden ejecutar desde el documento main.py. La URL utilizada debe cambiarse dependiendo la que proporcione la plataforma de educacion TripleTen debido a que es de un servidor que debe renovarse cada 2 horas
